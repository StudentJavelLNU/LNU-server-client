## Requirements

* Java
* Maven

## Usage

Get commandline options

```shell
java -jar networking-1.0-SNAPSHOT.one-jar.jar
```

Start server

```shell
java -jar networking-1.0-SNAPSHOT.one-jar.jar -m server
```

Start client

```shell
java -jar networking-1.0-SNAPSHOT.one-jar.jar -m client
```
